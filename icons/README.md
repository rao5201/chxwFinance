# 应用图标说明

本目录用于存放茶海虾王@金融交易所看板平台的应用图标。

## 需要创建的图标文件

根据 `manifest.json` 文件的配置，需要创建以下图标文件：

1. **icon-192x192.png** - 192x192像素的应用图标
2. **icon-512x512.png** - 512x512像素的应用图标

## 图标设计要求

- **风格**：现代金融风格，与网站整体设计一致
- **颜色**：使用金色(#E6AF2E)和深蓝色(#0A192F)作为主色调
- **内容**：可以包含金融相关元素，如金币、图表、交易所标志等
- **背景**：建议使用透明背景或与网站背景一致的深蓝色

## 创建方法

1. **使用设计工具**：
   - Adobe Photoshop
   - Adobe Illustrator
   - Canva
   - Figma

2. **在线生成**：
   - [PWA Icon Generator](https://www.pwabuilder.com/imageGenerator)
   - [Favicon.io](https://favicon.io/favicon-generator/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)

3. **尺寸要求**：
   - 192x192像素：用于移动设备主屏幕
   - 512x512像素：用于安装提示和应用商店

## 放置位置

将生成的图标文件直接放置在本目录中，确保文件名与 `manifest.json` 中指定的名称一致。

## 验证

图标创建完成后，可以通过以下方式验证：

1. 打开网站，检查浏览器地址栏是否显示正确的图标
2. 尝试安装PWA应用，检查安装过程中是否显示正确的图标
3. 安装完成后，检查设备主屏幕上的应用图标是否正确显示

## 注意事项

- 图标文件大小建议控制在50KB以内，以确保快速加载
- 确保图标在不同设备和背景下都能清晰显示
- 定期检查图标是否需要更新，以保持与品牌形象的一致性
