# 茶海虾王@金融交易所看板平台 - 网站文件汇总

## 项目概览

茶海虾王金融交易所看板平台是一个专业的金融交易分析工具，集成了AI智能分析、实时行情、技术指标、投资组合管理等功能，为投资者和交易者提供全方位的金融分析工具。

## 核心功能

- **AI智能分析**：基于机器学习的智能价格预测、模式识别、情绪分析和风险评估
- **技术指标**：50+种技术指标，包括MA、RSI、MACD、布林带、KDJ、斐波那契等
- **实时行情**：股票、加密货币、外汇实时行情推送，WebSocket实时数据更新
- **投资组合**：多维度资产管理、收益分析、风险监控、自动再平衡
- **量化交易**：策略回测、自动交易、多因子模型、机器学习策略
- **本地AI模型**：支持Ollama和LM Studio本地模型，保护数据隐私
- **Choice数据量化接口**：多语言支持的量化数据接口，为专业投资者提供丰富的金融数据
- **机器人系统**：跨平台、可扩展、高性能的机器人系统
