**版本**: v1.0.0

## 一、客服工作台概述

### 1.1 工作台功能
客服工作台是处理用户咨询、投诉、建议的主要工作界面，包含以下核心功能：
- 在线客服聊天
- 工单管理
- 用户查询
- 问题知识库
- 数据统计

### 1.2 登录方式
1. 访问后台管理系统：`https://chahaixiawang.com/admin-dashboard.html`
2. 使用客服账号登录（非管理员账号）
3. 进入"客服工作台"模块

---

## 二、在线客服聊天

### 2.1 接入用户咨询
1. 登录后自动进入"在线客服"状态
2. 系统会自动分配待处理的咨询
3. 点击用户头像或消息提醒查看详情
4. 在聊天窗口中回复用户问题

### 2.2 聊天操作
| 操作 | 说明 | 快捷键 |
|------|------|--------|
| 发送消息 | 输入内容后按Enter发送 | Enter |
| 换行 | 需要换行时按Shift+Enter | Shift+Enter |
| 快捷回复 | 点击快捷回复按钮选择常用语 | / |
| 转接 | 将用户转接给其他客服 | Ctrl+T |
| 结束会话 | 问题解决后结束会话 | Ctrl+E |

### 2.3 快捷回复设置
1. 点击"设置" → "快捷回复"
2. 添加常用回复语：
   - 问候语："您好，欢迎咨询茶海虾王金融平台"
   - 常见问题："关于账户问题，请您提供注册手机号"
   - 结束语："感谢您的咨询，如有其他问题请随时联系"
3. 保存后可在聊天窗口快速调用

---

## 三、工单管理

### 3.1 工单创建
当用户问题需要后续跟进时，创建工单：
1. 在聊天窗口点击"创建工单"
2. 填写工单信息：
   - 工单类型：咨询/投诉/建议/技术问题
   - 优先级：紧急/高/中/低
   - 问题描述：详细描述用户问题
   - 用户信息：自动填充
3. 提交工单

### 3.2 工单处理流程
```
待处理 → 处理中 → 待确认 → 已解决 → 已关闭
```

### 3.3 工单操作
| 状态 | 操作 | 说明 |
|------|------|------|
| 待处理 | 接单 | 认领工单开始处理 |
| 处理中 | 更新进度 | 记录处理过程和进展 |
| 待确认 | 联系用户 | 通知用户确认解决方案 |
| 已解决 | 关闭工单 | 用户确认后关闭 |
| 已关闭 | 查看记录 | 可查看历史工单 |

### 3.4 工单统计
- **我的工单**：显示当前客服处理的工单
- **待处理**：显示未分配的工单
- **已解决**：显示本月已解决的工单
- **满意度**：显示用户评价和满意度

## 四、用户查询

### 4.1 查询方式
1. **按手机号查询**：输入用户注册手机号
2. **按用户名查询**：输入用户昵称或用户名
3. **按用户ID查询**：输入系统分配的用户ID

### 4.2 用户信息查看
查询后可查看以下信息：
- **基本信息**：注册时间、账户状态、实名认证
- **资产信息**：账户余额、持有资产、交易记录
- **操作记录**：登录记录、交易记录、充值提现
- **工单历史**：历史咨询和投诉记录

### 4.3 用户操作
- **重置密码**：协助用户重置登录密码
- **解锁账户**：处理账户锁定问题
- **修改信息**：协助修改用户资料
- **备注**：添加用户备注信息

## 五、问题知识库

### 5.1 知识库分类
| 分类 | 内容 |
|------|------|
| 账户问题 | 注册、登录、密码、实名认证 |
| 交易问题 | 买入、卖出、交易规则、手续费 |
| 资产问题 | 充值、提现、资产查询、转账 |
| 技术问题 | 系统故障、APP问题、网络问题 |
| 投诉建议 | 投诉处理、建议收集、满意度调查 |

### 5.2 知识库使用
1. 在聊天窗口输入关键词搜索
2. 系统自动匹配相关知识点
3. 点击知识点查看详细解答
4. 可直接发送给用户或复制内容

### 5.3 知识库维护
- 发现新问题及时记录
- 定期更新过时内容
- 补充常见问题解答
- 优化搜索关键词

## 六、常见问题处理

### 6.1 账户问题

#### 无法登录
1. 确认用户名和密码是否正确
2. 检查账户是否被锁定
3. 协助重置密码
4. 如仍无法解决，转技术部门

#### 实名认证失败
1. 检查身份证照片是否清晰
2. 确认姓名和身份证号是否一致
3. 建议重新上传或联系人工审核

### 6.2 交易问题

#### 交易失败
1. 检查账户余额是否充足
2. 确认交易时间是否在交易时段
3. 查看是否有未完成的订单
4. 如系统问题，记录并上报

#### 提现不到账
1. 确认提现申请是否成功
2. 检查银行卡信息是否正确
3. 查询银行处理进度
4. 超过24小时未到账，提交工单跟进

### 6.3 技术问题

#### APP闪退
1. 建议清理缓存
2. 检查是否为最新版本
3. 建议重新安装APP
4. 记录设备型号和系统版本

#### 页面加载慢
1. 检查网络连接
2. 建议切换网络环境
3. 清除浏览器缓存
4. 如服务器问题，上报技术部门

## 七、服务规范

### 7.1 服务态度
- 礼貌用语："您好"、"请"、"谢谢"、"再见"
- 耐心倾听：不打断用户，理解用户诉求
- 积极主动：主动提供帮助，不推诿责任
- 专业解答：准确回答问题，不随意承诺

### 7.2 响应时间
| 场景 | 响应时间 |
|------|---------|
| 首次响应 | ≤ 30秒 |
| 问题回复 | ≤ 2分钟 |
| 复杂问题 | ≤ 5分钟 |
| 工单处理 | 按优先级处理 |

### 7.3 服务禁语
- ❌ "不知道"、"不清楚"
- ❌ "这是你的问题"
- ❌ "系统就是这样"
- ❌ "找别人去"
- ✅ "我帮您查询一下"
- ✅ "这个问题我来帮您解决"
- ✅ "系统正在优化中"
- ✅ "我为您转接专业同事"

## 九、应急处理

### 9.1 系统故障
1. 立即上报技术部门
2. 记录故障现象和时间
3. 安抚用户情绪
4. 告知预计恢复时间
5. 故障恢复后回访用户

### 9.2 重大投诉
1. 立即上报主管
2. 详细记录投诉内容
3. 创建高优先级工单
4. 协助相关部门处理
5. 跟进处理结果并反馈用户

### 9.3 安全事件
1. 发现异常立即上报
2. 保护用户信息安全
3. 协助安全部门调查
4. 配合提供相关记录
