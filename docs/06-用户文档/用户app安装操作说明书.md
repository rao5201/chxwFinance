# 茶海虾王@金融交易所看板平台
## 用户App安装操作说明书

## 一、安装准备

### 1.1 系统要求

| 设备类型 | 最低系统版本 | 推荐浏览器 |
|---------|------------|----------|
| Windows | Windows 10+ | Chrome 80+、Edge 80+ |
| macOS | macOS 10.13+ | Chrome 80+、Safari 13+ |
| iOS | iOS 13+ | Safari |
| Android | Android 6.0+ | Chrome 80+ |

### 1.2 网络要求
- 需要稳定的网络连接（首次安装和数据同步）
- 支持Wi-Fi和移动数据网络
- 建议使用5G/4G网络以获得最佳体验

## 二、安装步骤

### 2.1 Windows设备安装

#### Chrome浏览器
1. **打开网站**：在Chrome浏览器中访问 `http://localhost:8000`（线上环境请使用实际域名）
2. **触发安装提示**：
   - 方法一：等待浏览器自动弹出安装提示
   - 方法二：点击地址栏右侧的"安装"图标（拼图形状）
3. **确认安装**：在弹出的对话框中点击"安装"
4. **完成安装**：应用将自动安装到桌面，生成"茶海虾王金融"图标

#### Edge浏览器
1. **打开网站**：在Edge浏览器中访问 `http://localhost:8000`
2. **触发安装**：点击地址栏右侧的"应用"图标
3. **选择安装**：在下拉菜单中选择"安装此站点作为应用"
4. **确认安装**：在弹出的对话框中点击"安装"

### 2.2 macOS设备安装

#### Chrome浏览器
1. **打开网站**：在Chrome浏览器中访问 `http://localhost:8000`
2. **触发安装**：点击地址栏右侧的"安装"图标
3. **确认安装**：在弹出的对话框中点击"安装"
4. **完成安装**：应用将出现在Launchpad和 Dock中

#### Safari浏览器
1. **打开网站**：在Safari浏览器中访问 `http://localhost:8000`
2. **触发安装**：点击顶部菜单栏的"文件" > "添加到Dock"
3. **确认安装**：在弹出的对话框中点击"添加"

### 2.3 iOS设备安装

#### Safari浏览器
1. **打开网站**：在Safari浏览器中访问 `http://localhost:8000`
2. **触发安装**：点击底部的"分享"按钮（方形带箭头图标）

### 2.4 Android设备安装

#### Chrome浏览器
1. **打开网站**：在Chrome浏览器中访问 `http://localhost:8000`
2. **触发安装**：
   - 方法一：等待浏览器自动弹出安装提示
   - 方法二：点击浏览器菜单（三点图标） > "添加到主屏幕"
3. **确认安装**：在弹出的对话框中点击"安装"
4. **完成安装**：应用将安装到主屏幕

## 四、与网站对接同步机制

### 4.1 数据同步原理

**实时同步机制**：
- 应用采用WebSocket技术实现与服务器的实时数据传输
- 所有用户操作会立即同步到服务器
- 服务器数据更新会实时推送到应用

**离线同步机制**：
- 应用会缓存关键数据到本地存储
- 离线状态下的操作会被记录并在网络恢复后自动同步
- 确保数据一致性和完整性

### 4.2 同步数据类型

| 数据类型 | 同步方式 | 同步频率 | 离线支持 |
|---------|---------|---------|----------|
| 用户认证信息 | 实时同步 | 登录/登出时 | ✅ 支持 |
| 钱包余额 | 实时同步 | 交易后立即 | ✅ 支持 |
| 交易记录 | 实时同步 | 交易完成后 | ✅ 支持 |
| 个人资产 | 实时同步 | 资产变动时 | ✅ 支持 |
| 市场数据 | 定时同步 | 每30秒 | ✅ 支持（缓存） |
| AI分析结果 | 实时同步 | 分析完成后 | ❌ 需要网络 |

### 4.3 同步状态指示

- **在线状态**：应用顶部显示绿色连接图标
- **同步中**：应用顶部显示蓝色同步图标
- **离线状态**：应用顶部显示灰色离线图标
- **同步错误**：应用顶部显示红色错误图标并提示错误信息

### 4.4 手动同步

如需手动触发数据同步：
1. 点击应用右上角的用户头像
2. 在下拉菜单中选择"同步数据"
3. 等待同步完成提示

## 六、常见安装问题与解决方案

### 6.1 安装失败

**问题**：点击安装后无反应或安装失败
**解决方案**：
1. 检查浏览器版本是否符合要求
2. 清除浏览器缓存后重试
3. 尝试使用其他支持PWA的浏览器
4. 确保网络连接稳定

### 6.2 安装后无法启动

**问题**：应用安装成功但无法启动
**解决方案**：
1. 检查设备网络连接
2. 重新启动设备后重试
3. 卸载应用后重新安装
4. 检查设备存储空间是否充足

### 6.3 同步问题

**问题**：应用数据与网站不同步
**解决方案**：
1. 检查网络连接状态
2. 手动触发数据同步
3. 重新登录账号
4. 清除应用缓存后重试

### 6.4 权限问题

**问题**：应用无法发送通知或访问存储
**解决方案**：
1. 在设备设置中检查应用权限
2. 重新授权必要的权限
3. 确保设备系统权限设置正确
