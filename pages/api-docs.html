三、认证API

POST /auth/register

参数 类型 必填 描述
username string 是 用户名
email string 是 邮箱地址
password string 是 密码

POST /auth/login
用户登录

password string 是 密码

十一、本地AI模型API

GET /finance/local-ai/config
获取本地AI模型配置信息
GET /finance/local-ai/test/:provider
测试本地AI模型连接（provider: ollama 或 lmstudio）
POST /finance/local-ai/generate/:provider
使用本地AI模型生成文本
参数 类型 必填 描述
prompt string 是 输入提示词
model string 否 模型名称
options object 否 生成选项（temperature, maxTokens等）
POST /finance/local-ai/analyze/:provider
使用本地AI模型进行金融分析
参数 类型 必填 描述
type string 是 分析类型（marketTrend, riskAssessment, portfolioOptimization, financialAnalysis, newsSentiment）
data object 是 分析数据

支持的本地模型提供商

Ollama

  • 安装: https://ollama.com/download
  • 默认端口: 11434
  • 默认模型: llama2
  • 运行: ollama serve
  • 支持D盘安装: 是
  • 测试脚本: node scripts/test-ollama.js

LM Studio

  • 安装: https://lmstudio.ai/
  • 默认端口: 1234
  • API格式: OpenAI兼容
  • 运行: 在LM Studio中启动本地服务器

十二、错误码说明

状态码 含义 说明
200 成功 请求成功处理
201 创建成功 资源创建成功
400 请求参数错误 请求参数不完整或格式错误
401 未授权 缺少认证令牌或令牌无效
403 禁止访问 权限不足或账户被禁用
404 资源不存在 请求的资源不存在
500 服务器错误 服务器内部错误

十二、认证说明

所有需要认证的API都需要在请求头中携带JWT令牌：

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

令牌有效期为7天，过期后需要使用刷新令牌获取新的访问令牌。
