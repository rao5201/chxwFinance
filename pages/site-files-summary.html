项目概览

茶海虾王金融交易所看板平台是一个专业的金融交易分析工具，集成了AI智能分析、实时行情、技术指标、投资组合管理等功能，为投资者和交易者提供全方位的金融分析工具。

核心功能

AI智能分析

基于机器学习的智能价格预测、模式识别、情绪分析和风险评估

技术指标

50+种技术指标，包括MA、RSI、MACD、布林带、KDJ、斐波那契等

实时行情

股票、加密货币、外汇实时行情推送，WebSocket实时数据更新

投资组合

多维度资产管理、收益分析、风险监控、自动再平衡

本地AI模型

支持Ollama和LM Studio本地模型，保护数据隐私

Choice数据量化接口

多语言支持的量化数据接口，为专业投资者提供丰富的金融数据

API端点

认证API

  • POST /api/auth/register - 注册
  • POST /api/auth/login - 登录
  • POST /api/auth/refresh - 刷新令牌
  • POST /api/auth/logout - 登出
  • GET /api/auth/me - 获取当前用户

市场API

  • GET /api/market/overview - 市场行情
  • GET /api/market/price/:symbol - 资产价格
  • GET /api/market/kline/:symbol - K线数据
  • GET /api/market/pairs - 交易对
  • GET /api/market/orderbook/:symbol - 市场深度
  • GET /api/market/trades/:symbol - 交易历史

金融分析API

  • POST /api/analysis/technical - 技术分析
  • POST /api/analysis/fundamental - 基本面分析
  • POST /api/analysis/risk - 风险管理
  • POST /api/analysis/portfolio - 投资组合分析
  • POST /api/analysis/sentiment - 市场情绪分析

AI分析API

  • POST /api/ai/analysis/market-trend - 市场趋势分析
  • POST /api/ai/analysis/risk-assessment - 投资风险评估
  • POST /api/ai/analysis/portfolio-optimization - 投资组合优化
  • POST /api/ai/analysis/market-prediction - 市场走势预测
  • POST /api/ai/analysis/financial-analysis - 财务报表分析
  • POST /api/ai/analysis/news-sentiment - 新闻情绪分析

本地AI模型API

  • GET /api/finance/local-ai/config - 获取配置信息
  • GET /api/finance/local-ai/test/:provider - 测试连接
