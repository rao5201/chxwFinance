## 遇到的问题和解决方案

### 1. 数据库连接异常
- **问题**：MySQL 连接失败，表结构初始化失败
- **解决方案**：
  - 实现故障自动切换到内存数据库
  - 修改表结构初始化逻辑，改为检查现有表结构
  - 优化数据库连接配置，移除无效参数

### 2. 端口冲突
- **问题**：服务器端口被占用
- **解决方案**：
  - 多次修改端口号（3005 → 3008 → 3009 → 3010 → 3011 → 3012 → 3013 → 3014 → 3015）
  - 最终确定使用端口 3015

### 3. 第三方 API 调用失败
- **问题**：腾讯文档和有道云笔记 API 调用失败
- **解决方案**：
  - 实现详细的错误处理和日志记录
  - 实现故障降级机制，返回模拟数据
  - 优化 API 配置，使用正确的 API 地址和认证方式

### 4. 权限限制
- **问题**：MySQL 用户无创建表权限
- **解决方案**：
  - 改为检查现有表结构，不再尝试创建表
  - 实现内存数据库作为备用存储
